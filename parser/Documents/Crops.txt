<h1 class="firstHeading" id="firstHeading" lang="en">Crop</h1>

<p>A <b>crop</b> is "a plant or animal product that can be grown and harvested extensively for profit or subsistence."<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Crop may refer either to the harvested parts or to the harvest in a more refined state (<a href="/wiki/Husk" title="Husk">husked</a>, shelled, etc.). Most crops are cultivated in <a href="/wiki/Agriculture" title="Agriculture">agriculture</a> or <a href="/wiki/Aquaculture" title="Aquaculture">aquaculture</a>. A <b>crop</b> is usually expanded to include macroscopic <a href="/wiki/Fungus" title="Fungus">fungus</a> (e.g. <a href="/wiki/Mushroom" title="Mushroom">mushrooms</a>), or <a href="/wiki/Algae" title="Algae">alga</a> (<a href="/wiki/Algaculture" title="Algaculture">algaculture</a>).</p>
<p>Most crops are <a href="/wiki/Harvest" title="Harvest">harvested</a> as <a href="/wiki/Food" title="Food">food</a> for humans or <a href="/wiki/Livestock" title="Livestock">livestock</a> (<a href="/wiki/Fodder" title="Fodder">fodder</a> crops). Some crops are gathered from the wild (including <a href="/wiki/Intensive_gathering" title="Intensive gathering">intensive gathering</a>, e.g. <a href="/wiki/Ginseng" title="Ginseng">ginseng</a>).</p>
<p>Important non-food crops include horticulture, floriculture and industrial crops. <a href="/wiki/Horticulture" title="Horticulture">Horticulture</a> crops include plants used for other crops (e.g. <a class="mw-redirect" href="/wiki/Fruit_trees" title="Fruit trees">fruit trees</a>). <a href="/wiki/Floriculture" title="Floriculture">Floriculture</a> crops include bedding plants, houseplants, flowering garden and pot plants, cut cultivated greens, and cut flowers. <a href="/wiki/Industrial_crop" title="Industrial crop">Industrial crops</a> are produced for <a href="/wiki/Clothing" title="Clothing">clothing</a> (<a href="/wiki/Fiber_crop" title="Fiber crop">fiber crops</a>), <a href="/wiki/Biofuel" title="Biofuel">biofuel</a> (<a href="/wiki/Energy_crop" title="Energy crop">energy crops</a>, <a href="/wiki/Algae_fuel" title="Algae fuel">algae fuel</a>), or <a href="/wiki/Medicine" title="Medicine">medicine</a> (<a href="/wiki/Medicinal_plants" title="Medicinal plants">medicinal plants</a>).</p>
<p><a href="/wiki/Animal" title="Animal">Animals</a> and <a href="/wiki/Microorganism" title="Microorganism">microbes</a> (<a class="mw-redirect" href="/wiki/Fungi" title="Fungi">fungi</a>, <a href="/wiki/Bacteria" title="Bacteria">bacteria</a> or <a href="/wiki/Virus" title="Virus">viruses</a>) are rarely referred to as crops. Animals raised for <a href="/wiki/Human" title="Human">human</a> or <a href="/wiki/Animal" title="Animal">animal</a> consumption are referred to as <a href="/wiki/Livestock" title="Livestock">livestock</a> and microbes as <a href="/wiki/Microbiological_culture" title="Microbiological culture">microbiological cultures</a>. Microbes are not typically grown for food itself, but are rather used to alter food (e.g., producing <a href="/wiki/Citric_acid" title="Citric acid">citric acid</a>, <a href="/wiki/Fermentation_in_food_processing" title="Fermentation in food processing">fermenting</a> <a href="/wiki/Yogurt" title="Yogurt">yogurt</a>, <a href="/wiki/Soy_sauce" title="Soy sauce">soy sauce</a>, or <a href="/wiki/Sauerkraut" title="Sauerkraut">sauerkraut</a>).</p>

<p>The importance of a crop varies greatly by region. Globally, the following crops contribute most to human food supply (values of kcal/person/day for 2013 given in parentheses): <a href="/wiki/Rice" title="Rice">Rice</a> (541 kcal), <a href="/wiki/Wheat" title="Wheat">wheat</a> (527 kcal), <a href="/wiki/Sugarcane" title="Sugarcane">sugarcane</a> and other sugar crops (200 kcal), <a href="/wiki/Maize" title="Maize">maize</a> (corn) (147 kcal), <a href="/wiki/Soybean_oil" title="Soybean oil">soybean oil</a> (82 kcal), other <a href="/wiki/Vegetable" title="Vegetable">vegetables</a> (74 kcal), <a href="/wiki/Potato" title="Potato">potatoes</a> (64 kcal), <a href="/wiki/Palm_oil" title="Palm oil">palm oil</a> (52 kcal), <a href="/wiki/Cassava" title="Cassava">cassava</a> (37 kcal), <a href="/wiki/Legume" title="Legume">legume</a> pulses (37 kcal), <a class="mw-redirect" href="/wiki/Sunflowerseed" title="Sunflowerseed">sunflowerseed</a> oil (35 kcal), <a class="mw-redirect" href="/wiki/Rape_oil" title="Rape oil">rape</a> and <a href="/wiki/Mustard_oil" title="Mustard oil">mustard oil</a> (34 kcal), other <a href="/wiki/Fruit" title="Fruit">fruits</a>, (31 kcal), <a href="/wiki/Sorghum" title="Sorghum">sorghum</a> (28 kcal), <a href="/wiki/Millet" title="Millet">millet</a> (27 kcal), <a href="/wiki/Groundnut" title="Groundnut">groundnuts</a> (25 kcal), <a href="/wiki/Bean" title="Bean">beans</a> (23 kcal), <a href="/wiki/Sweet_potato" title="Sweet potato">sweet potatoes</a> (22 kcal), <a href="/wiki/Banana" title="Banana">bananas</a> (21 kcal), <a href="/wiki/Nut_(fruit)" title="Nut (fruit)">various nuts</a> (16 kcal), <a class="mw-redirect" href="/wiki/Soyabean" title="Soyabean">soyabeans</a> (14 kcal), <a href="/wiki/Cottonseed_oil" title="Cottonseed oil">cottonseed oil</a> (13 kcal), <a class="mw-redirect" href="/wiki/Groundnut_oil" title="Groundnut oil">groundnut oil</a> (13 kcal), <a href="/wiki/Yam_(vegetable)" title="Yam (vegetable)">yams</a> (13 kcal).<sup class="reference" id="cite_ref-FAOstats_2-0"><a href="#cite_note-FAOstats-2">[2]</a></sup> Note that many of the globally apparently minor crops are regionally very important. For example in Africa, roots &amp; tubers dominate with 421 kcal/person/day, and sorghum and millet contribute 135 kcal and 90 kcal, respectively.<sup class="reference" id="cite_ref-FAOstats_2-1"><a href="#cite_note-FAOstats-2">[2]</a></sup></p>

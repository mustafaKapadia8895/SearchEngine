<h1 class="firstHeading" id="firstHeading" lang="en">Wave height</h1>
<div class="mw-body-content" id="bodyContent">

<p>In <a href="/wiki/Fluid_dynamics" title="Fluid dynamics">fluid dynamics</a>, the <b>wave height</b> of a <a class="mw-redirect" href="/wiki/Ocean_surface_wave" title="Ocean surface wave">surface wave</a> is the difference between the <a href="/wiki/Elevation" title="Elevation">elevations</a> of a <a href="/wiki/Crest_(physics)" title="Crest (physics)">crest</a> and a neighbouring <a class="mw-redirect" href="/wiki/Trough_(physics)" title="Trough (physics)">trough</a>.<sup class="reference" id="cite_ref-Kinsman_38_1-0"><a href="#cite_note-Kinsman_38-1">[1]</a></sup> Wave height is a term used by <a class="mw-redirect" href="/wiki/Mariner" title="Mariner">mariners</a>, as well as in <a href="/wiki/Coastal_engineering" title="Coastal engineering">coastal</a>, <a href="/wiki/Offshore_construction" title="Offshore construction">ocean</a> and <a class="mw-redirect" href="/wiki/Naval_engineering" title="Naval engineering">naval engineering</a>.</p>
<p>At sea, the term <a href="/wiki/Significant_wave_height" title="Significant wave height">significant wave height</a> is used as a means to introduce a well-defined and standardized <a href="/wiki/Statistic" title="Statistic">statistic</a> to denote the characteristic height of the <a class="mw-redirect" href="/wiki/Random" title="Random">random</a> waves in a <a href="/wiki/Sea_state" title="Sea state">sea state</a>. It is defined in such a way that it more–or–less corresponds to what a <a class="mw-redirect" href="/wiki/Mariner" title="Mariner">mariner</a> observes when estimating visually the average wave height.</p>

<ul>
<li>In random waves at sea, when the surface elevations are measured with a <a href="/wiki/Weather_buoy" title="Weather buoy">wave buoy</a>, the <i>individual wave height</i> <i>H</i><sub>m</sub> of each individual wave—with an <a href="/wiki/Integer" title="Integer">integer</a> label <i>m</i>, running from 1 to <i>N</i>, to denote its position in a sequence of <i>N</i> waves—is the difference in elevation between a wave crest and trough in that wave. For this to be possible, it is necessary to first split the measured <a href="/wiki/Time_series" title="Time series">time series</a> of the surface elevation into individual waves. Commonly, an individual wave is denoted as the <a class="mw-redirect" href="/wiki/Time_interval" title="Time interval">time interval</a> between two successive downward-crossings through the <a href="/wiki/Average" title="Average">average</a> surface elevation (upward crossings might also be used). Then the individual wave height of each wave is again the difference between maximum and minimum elevation in the time interval of the wave under consideration.<sup class="reference" id="cite_ref-Holt_24_28_2-0"><a href="#cite_note-Holt_24_28-2">[2]</a></sup></li>
<li><i><a href="/wiki/Significant_wave_height" title="Significant wave height">Significant wave height</a></i> <i>H</i><sub>1/3</sub>, or <i>H</i><sub><i>s</i></sub> or <i>H</i><sub><i>sig</i></sub>, in the <i><a href="/wiki/Time_domain" title="Time domain">time domain</a></i>, is defined as the average height of the one-third part of the measured waves—which are <i>N</i> in number—having the largest wave heights:<sup class="reference" id="cite_ref-Holt_24_28_2-1"><a href="#cite_note-Holt_24_28-2">[2]</a></sup></li>
</ul>
